# AI Resume – Job Matching System

This project extracts information from resumes and matches them with job descriptions using Natural Language Processing.

## Features

- Resume parsing and information extraction
- Skill extraction from resumes
- Resume and job description matching
- Similarity scoring using NLP

## Tech Stack

- Python
- Natural Language Processing
- Scikit-learn
- pyresparser

## How it Works

1. Upload a resume
2. Extract skills and information
3. Compare with job description
4. Calculate similarity score
